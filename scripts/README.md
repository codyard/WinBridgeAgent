# 脚本说明

快速入口：
- 构建：`./scripts/build.sh`
- 发布：`./scripts/release.sh`
- 一键发布：`./scripts/auto-release.sh`
- 初始化：`./scripts/init-repos.sh`
- 安装：`./scripts/install.sh`
- 运行全部测试：`./scripts/run_all_v030_tests.sh`

脚本列表：
- `auto-release.sh`: 一键发布流程
- `build.sh`: 交叉编译 x64/x86
- `init-repos.sh`: 初始化仓库与子模块/依赖
- `install.sh`: 一键安装脚本
- `release.sh`: 打包发布产物
- `run_all_v030_tests.sh`: 运行 v0.3.0 全部测试
- `test_all_features.sh`: 全功能测试
- `test_auth.sh`: 认证测试
- `test_clipboard.sh`: 剪贴板测试
- `test_dashboard.sh`: Dashboard 测试
- `test_file_operations.sh`: 文件操作测试
- `test_file_ops.sh`: 文件操作测试（备用）
- `test_mcp.sh`: MCP 协议测试
- `test_power_management.sh`: 电源管理测试
- `test_process_management.sh`: 进程管理测试
- `test_screenshot.sh`: 截图测试
- `test_security.sh`: 安全测试
- `test_windows_processes.sh`: Windows 进程测试
