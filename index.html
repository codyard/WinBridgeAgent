ğŸš€ Quick Start

1. Download and Install

Download the latest release from GitHub Releases and run the executable:

  • WinBridgeAgent-x64.exe for 64-bit Windows
  • WinBridgeAgent-x86.exe for 32-bit Windows
  • WinBridgeAgent-arm64.exe for ARM64 Windows

2. Configure MCP Client

For Claude Desktop, edit claude_desktop_config.json:

{
    "mcpServers": {
        "winbridgeagent": {
            "url": "http://<windows-ip>:35182",
            "transport": "http"
        }
    }
}

For Cursor, edit .cursor/mcp.json:

{
    "mcpServers": {
        "winbridgeagent": {
            "url": "http://<windows-ip>:35182",
            "transport": "http"
        }
    }
}

Replace <windows-ip> with your Windows PC's IP address (e.g., 192.168.1.100). Use localhost if the MCP client runs on the same machine.

ğŸ”’ Security

  • Listen Address: Use 0.0.0.0 for LAN access or 127.0.0.1 for local-only
  • Protected Processes: System-critical processes cannot be terminated
  • Protected Directories: System directories cannot be deleted
  • Audit Logging: All operations are logged for traceability
