# Resources Directory

This directory contains resource files for WinBridgeAgent.

## Files

- `config.template.json` - Configuration file template
- `icon.ico` - Application icon (TODO: Create proper icon)

## Icon Requirements

The icon should include the following sizes:

- 16x16 (托盘图标)
- 32x32 (托盘图标高 DPI)
- 48x48 (应用程序图标)
- 256x256 (高分辨率图标)

Icon states needed:

- Normal (green) - 服务运行中
- Stopped (gray) - 服务已停止
- Error (red) - 服务异常
- Warning (yellow) - License 即将过期
